Section 2.3 Fractions involving a variable

In this section, we will discuss fractions with a variable in the numerator, the denominator, or both. These fractions are really just functions, and as with all functions we need to restrict the values that the variable can take so that the function is defined. For example, \(\frac{1}{x}\) is undefined for \(x=0\text{.}\)

Addition, subtraction, multiplication, and division of fractions that involve a variable are all done in the same way they were with real numbers, except that, in addition to the restrictions from above, we also need to restrict the values that the variable can take so that the operations are defined as well. For example, \(\frac{1}{x}\) and \(\frac{x+2}{x-1}\) are defined only for \(x\neq0,1\text{,}\) and \(\frac{x+2}{x-1}=0\) for \(x=-2\text{,}\) so \(\frac{1}{x}\div\frac{x+2}{x-1}\) only makes sense for \(x\neq-2,0,1\text{.}\)

Example 2.3.1.

Compute and write the answer in lowest terms.

  • \(\frac{x+4}{x-3} + \frac{2x+8}{x-3} = \frac{x+4+2x+8}{x-3} =\frac{3x+12}{x-3} =\frac{3(x+4)}{x-3}\text{.}\)

  • \(\frac{1}{x-2} + \frac{4}{x-5} = \frac{1 \cdot (x-5)}{(x-2) \cdot (x-5)}+ \frac{4\cdot (x-2)}{(x-5) \cdot (x-2)} = \frac{x-5}{(x-2) (x-5)} + \frac{4x-8}{(x-5) (x-2)} = \frac{x-5+4x-8}{(x-5) (x-2)} = \frac{5x-13}{(x-5) (x-2)}\text{.}\)

  • \(\frac{2}{x+5} - \frac{4}{x-6} = \frac{2 \cdot (x-6)}{(x+5) \cdot (x-6)} - \frac{4 \cdot (x+5)}{(x-6) \cdot (x+5)} = \frac{2x-12}{(x+5)(x-6)} - \frac{4x+20}{(x+5)(x+6)} = \frac{2x-12 - (4x+20)}{(x+5)(x+6)} = \frac{-2x-32}{(x+5)(x+6)} \\ = \frac{-2(x+16)}{(x+5)(x+6)}\text{.}\)

Sometimes when common factors are cancelled, the collection of vales for which the expression makes sense seems to change. It shouldn't. For example \(\frac{2x}{x(x-1)}\) is defined for all real numbers \(x \neq 0, 1\text{,}\) but \(\frac{2}{(x-1)}\) is defined for all real numbers \(x \neq 1\text{.}\) Since two expressions can only be equal for values of \(x\) where they are both defined, in this case we can only say \(\frac{2x}{x(x-1)} = \frac{2}{(x-1)}\) if \(x\) is restricted to be neither 0 nor 1.

Example 2.3.2.

Compute \(\frac{5x+9}{4x+16} \cdot \frac{9x+36}{2x-3}\text{.}\) Write the answer in lowest terms.

It is clear that we need \(x \neq -4\) for the first fraction to be defined, and \(x \neq \frac{3}{2}\) for the second fraction to be defined. Now:

\begin{align*} \frac{5x+9}{4x+16} \cdot \frac{9x+36}{2x-3} \amp =\amp \frac{5x+9}{4(x+4)} \cdot \frac{9(x+4)}{2x-3}\\ \amp =\amp \frac{(5x+9)9(x+4)}{4(x+4)(2x-3)}\\ \amp =\amp \frac{9(5x+9)}{ 4(2x-3)}, x\neq -4\text{.} \end{align*}

We noted the restriction \(x \neq -4\) when the factor \((x+4)\) was cancelled because, although it is not required for the final fraction to be defined, it is required so that all expressions involved in the equation are defined for the same set of values.

Example 2.3.3.

Compute and write the answer in lowest terms.

  • \(\frac{15x-5}{8x-4} \div \frac{10x+25}{6x-3} = \frac{15x-5}{8x-4} \cdot \frac{6x-3}{10x+25} = \frac{5(3x-1)}{4(2x-1)} \cdot \frac{3(2x-1)}{5(2x+1)} = \frac{5(3x-1)3(2x-1)}{4(2x-1)5(2x+1)} = \frac{3(3x-1)}{4(2x+1)}, x \neq \frac{1}{2}\text{.}\)

  • \(\frac{4}{x^2+3x+2} + \frac{5}{x+2} = \frac{4}{(x+2)(x+1)} + \frac{5}{x+2} = \frac{4}{(x+2)(x+1)} + \frac{5(x+1)}{(x+2)(x+1)} = \frac{4 +5(x+1)}{(x+2)(x+1)} \\ = \frac{4 +5x+5}{(x+2)(x+1)} = \frac{5x+9}{(x+2)(x+1)}\text{.}\)

Subsection 2.3.1 Practice Problems

Write the following expressions in lowest terms:

  1. \(\displaystyle \frac{3}{x+6} + \frac{8}{x+6}\)

  2. \(\displaystyle \frac{10x-3}{2x+5} + \frac{7x-4}{2x+5}\)

  3. \(\displaystyle \frac{5}{x+3} - \frac{8}{2x+6}\)

  4. \(\displaystyle \frac{x^2 -1}{x^2-x} \div \frac{3x^2 +x -2}{3x^2-17x+10}\)

  5. \(\displaystyle \frac{x+3}{5x-2} \cdot \frac{2x+1}{x-3}\)

Subsection 2.3.2 Solutions

  1. \(\displaystyle \frac{11}{x+6}\)

  2. \(\displaystyle \frac{17x-7}{2x+5}\)

  3. \(\displaystyle \frac{1}{x+3}\)

  4. \(\displaystyle \frac{x-5}{x}, x \neq -1, \frac{2}{3}, 1, 5\)

  5. \(\displaystyle \frac{(x+3) (2x+1)}{ (5x-2) (x-3)}\)
