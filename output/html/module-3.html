Checkpoint 3.6.10.

The equation of the graphed function is

  • \(\displaystyle y=-x^2 \)

  • \(\displaystyle y=x^2-5 \)

  • \(\displaystyle y=5-x^2 \)

  • \(\displaystyle y=5+x^2 \)

  • None of the above

Hint.

Look for evidence that will help eliminate wrong answers.

This graph has a maximum (highest \(y\) value), are there any answers you can eliminate because they don’t have maximums?

If more than one answer is left, can you eliminate any because you can see a point that is on the graph but does not satisfy the equation in that answer?

Answer.

\(\text{Choice 3}\)

Solution.

A look at the graph gives us the following information.

  1. The answer cannot be \(\( y=5+x^2 \)\) since this graph has a maximum at \((0,5)\) and \(\( y=5+x^2 \)\) has no maximum as \(x\) increases.

  2. The answer cannot be \(\( y=x^2-5 \)\) for two reasons.

First, when \(x=0\) we get \(\( y=x^2-5 \)=-5\) and we know from the graph than when \(x=0, y=5\)

Second, this graph has a maximum at \((0,5)\) and \(\( y=x^2-5 \)\) has no maximum as \(x\) increases.

  1. The answer cannot be \(\( y=-x^2 \)\) since when \(x=0, \( y=-x^2 \)=0\) and we know from the graph that when \(x=0, y=5\)

To make sure we might want to check the remaining answer, \(\( y=5-x^2 \)\text{.}\) It has the correct maximum at \((0,5\) and the graph looks like a parabola. If we were certain that the points at which the function corsses the \(x\)-axis were \((-\sqrt{5},0)\) and \((\sqrt{5},0)\) then we could be more sure, since just as the equation of a line can be found from knowing two points, the equation of a parabola can be found from knowing three points.

Checkpoint 3.6.13.

Tell whether the following ordered pairs are solutions to the given inequality.

\(y \le 4\)

  1. \((-8,4)\)

    • is a solution

    • is not a solution

  2. \((0,0)\)

    • is a solution

    • is not a solution

  3. \((-5,-6)\)

    • is a solution

    • is not a solution

  4. \((0,8)\)

    • is a solution

    • is not a solution

Answer. 1

\(\text{is a solution}\)

Answer. 2

\(\text{is a solution}\)

Answer. 3

\(\text{is a solution}\)

Answer. 4

\(\text{is not a solution}\)

Solution.

We substitute \(y\) values from each ordered pair into the inequality \(y \le 4\) and see whether the inequality is true.

  1. The ordered pair is \((-8,4)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} 4 \\ 4 \amp \stackrel{?}{\le} 4 \end{aligned}\)

    Since this inequality is true, \((-8,4)\) is a solution to the inequality.

  2. The ordered pair is \((0,0)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} 4 \\ 0 \amp \stackrel{?}{\le} 4 \end{aligned}\)

    Since this inequality is true, \((0,0)\) is a solution to the inequality.

  3. The ordered pair is \((-5,-6)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} 4 \\ -6 \amp \stackrel{?}{\le} 4 \end{aligned}\)

    Since this inequality is true, \((-5,-6)\) is a solution to the inequality.

  4. The ordered pair is \((0,8)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} 4 \\ 8 \amp \stackrel{?}{\le} 4 \end{aligned}\)

    Since this inequality is not true, \((0,8)\) is not a solution to the inequality.

Checkpoint 3.6.14.

Choose the graph that shows the solutions to \(y \geq {-3x}\text{.}\) On your own graphing paper, first graph the line (either dotted or solid), and then pick a test point to decide which side of the line should be shaded. Warning: if you skip the steps that you should be doing on paper, then you are not practicing what the exercise intends for your to practice.

The correct graph is graph

  • A

  • B

  • C

  • D
Answer.

\(\text{C}\)

Solution.

To graph the inequality \(y \geq -3x\text{,}\) we first graph the line \(y=-3x\text{.}\) You can use either the table method or slope-triangle method to locate two points on the line.

Note that the line is solid, because the inequality symbol is \(\geq\text{.}\)

Next, we need to decide which side to shade. Usually we substitute \(x=0\) and \(y=0\) into the inequality. However, for this problem, \((0,0)\) is on the line \(y = -3x\text{,}\) so we have to pick another point.

Let's pick the point \((0, 1)\text{.}\) We substitute \(x=0\) and \(y=1\) into \(y \geq -3x\text{,}\) and we have:

\begin{equation*} \begin{aligned}[t] y \amp \geq -3x \\ 1 \amp \geq -3 \cdot 0 \\ 1 \amp \geq 0 \end{aligned} \end{equation*}

This is true, so we shade the side with the point \((0, 1)\text{.}\)

We can see the correct answer is C.

Checkpoint 3.6.16.

Choose the graph that shows the solutions to \({4x+2y} \leq 8\text{.}\) On your own graphing paper, first graph the line (either dotted or solid), and then pick a test point to decide which side of the line should be shaded. Warning: if you skip the steps that you should be doing on paper, then you are not practicing what the exercise intends for your to practice.

The correct graph is graph

  • A

  • B

  • C

  • D
Answer.

\(\text{A}\)

Solution.

To graph the inequality \(4x+2y \leq 8\text{,}\) we first graph the line \(4x+2y=8\text{.}\) You can use either the table method or slope-triangle method to locate two points on the line.

Note that the line is solid, because the inequality symbol is \(\leq\text{.}\)

Next, we need to decide which side to shade. We choose \((0,0)\) as the test point. We substitute \(x=0\) and \(y=0\) into the inequality into \(4x+2y \leq 8\text{,}\) and we have:

\begin{equation*} \begin{aligned}[t] 4x+2y \amp \leq 8 \\ 4 \cdot 0 + 2 \cdot 0 \amp \leq 8 \\ 0 \amp \leq 8 \end{aligned} \end{equation*}

This is true, so we shade the side with the point \((0, 0)\text{.}\)

We can see the correct answer is A.
