Answer.

\(-1080\)

Solution.
SOLUTION If we go around -3 times, we make 3 full circles in the negative direction, which is \(-3\cdot 360^{\circ} = -1080^{\circ}\) .

Checkpoint 5.0.99.

A point \(P\) is often identified in mathematics using the notation \(P(x,y)\) to reference a point at coordinates \((x,y)\text{.}\)

If \(P(x,\frac{11}{13})\) is on the unit circle in quadrant II, then \(x =\) .

Hint.

The equation of the unit circle is

\begin{equation*} x^2+y^2 = 1 \end{equation*}

If you are given y, you can carefully solve for x.

Watch your algebra!

Answer.

\(-0.532938710021193\)

Solution.

SOLUTION:

Evaluating \(x^2+y^2 = 1\) for \(y = \frac{11}{13}\) gives

\begin{equation*} {(\frac{11}{13}})^2+x^2 = 1 \end{equation*}

Solving for x yields

\begin{equation*} x^2 = {(0.532938710021193)}^2 \end{equation*}

Since the point is in quadrant II, then x is negative and so \(x = -0.532938710021193\text{.}\)
