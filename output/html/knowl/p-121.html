Combining intervals. The set \(\{x : 2 < x < 5 \mbox{ or } 6 < x < 9\}\) is the set of all real numbers that are in the interval \((2,5)\) or are in the interval \((6,9)\text{.}\) We use the \(\cup\) symbol to signify the union of two sets.
