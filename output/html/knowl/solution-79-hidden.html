We substitute \(y\) values from each ordered pair into the inequality \(y \le -5\) and see whether the inequality is true.

  1. The ordered pair is \((0,0)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} -5 \\ 0 \amp \stackrel{?}{\le} -5 \end{aligned}\)

    Since this inequality is not true, \((0,0)\) is not a solution to the inequality.

  2. The ordered pair is \((5,-5)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} -5 \\ -5 \amp \stackrel{?}{\le} -5 \end{aligned}\)

    Since this inequality is true, \((5,-5)\) is a solution to the inequality.

  3. The ordered pair is \((0,10)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} -5 \\ 10 \amp \stackrel{?}{\le} -5 \end{aligned}\)

    Since this inequality is not true, \((0,10)\) is not a solution to the inequality.

  4. The ordered pair is \((-9,-9)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} -5 \\ -9 \amp \stackrel{?}{\le} -5 \end{aligned}\)

    Since this inequality is true, \((-9,-9)\) is a solution to the inequality.
