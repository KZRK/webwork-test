The numbers \(-1\) and \(5\) partition the set of other numbers in \(\mathbb{R}\) into three intervals: \((-\infty, -1)\text{,}\) \((-1, 5)\text{,}\) and \((5, \infty)\text{.}\) Because the expression can change sign only at \(-1\) or \(5\text{,}\) for each of these, either every number in the interval belongs to the set or no number in the interval belongs to the set. We need to determine which possibility occurs for each of them. We can do this by testing one number from each interval and then remembering that we know what happens with the expression at \(-1\) and \(5\text{.}\)
