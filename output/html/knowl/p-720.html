The properties of exponents are the same no matter whether the exponent is an integer, a rational number or a real number:

  • When exponential expressions with the same base are multiplied, simplify by adding the powers, i.e.,

    \begin{equation*} b^xb^y = b^{x+y}\text{.} \end{equation*}

  • When an exponential expression is itself raised to some power, simplify by multiplying the exponents, i.e.,

    \begin{equation*} \left(b^x\right))^y = b^{xy}\text{.} \end{equation*}
