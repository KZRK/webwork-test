Suppose \(f(x) = y\text{.}\) Then \(y = \frac{2x}{4x-1}\) and \(x \neq \frac{1}{4}\text{,}\) so that

\begin{equation*} g(y) = g\left(\frac{2x}{4x-1}\right) = \frac{\frac{2x}{4x-1}}{4\frac{2x}{4x-1} - 2} = \frac{\frac{2x}{4x-1}}{4\frac{2x}{4x-1} - 2\frac{4x-1}{4x-1}} = \frac{ \left(\frac{2x}{4x-1}\right) }{ \left(\frac{2}{4x-1}\right)} = x\text{.} \end{equation*}
