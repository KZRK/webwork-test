For the interval \((\frac{1}{3}, \frac{9 + 5\sqrt{3}}{6})\) we choose \(x=1\) as the test point. When, \(x = 1\) the expression evaluates to \(-1.1 \lt 0\text{,}\) so no points in this interval belong to the solution set. Since the expression equals zero when \(\frac{9 -+5\sqrt{3}}{6}\text{,}\) this point is not in the solution set.
