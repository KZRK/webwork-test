The graph of a function \(f(x)\) is the set of all points \((x, y)\) such that \(y = f(x)\text{.}\) It partitions the plane into two regions. The same inequality holds for all points in the same region. Hence what we need to do is sketch the graph of \(y = f(x)\text{,}\) pick a test point in each region and check which inequality holds, and then be careful about whether the points on the graph need to be included.
