Solution 3.0.68.1.

To graph the inequality \(2x+3y \leq -6\text{,}\) we first graph the line \(2x+3y=-6\text{.}\) You can use either the table method or slope-triangle method to locate two points on the line.

Note that the line is solid, because the inequality symbol is \(\leq\text{.}\)

Next, we need to decide which side to shade. We choose \((0,0)\) as the test point. We substitute \(x=0\) and \(y=0\) into the inequality into \(2x+3y \leq -6\text{,}\) and we have:

\begin{equation*} \begin{aligned}[t] 2x+3y \amp \leq -6 \\ 2 \cdot 0 + 3 \cdot 0 \amp \leq -6 \\ 0 \amp \leq -6 \end{aligned} \end{equation*}

This is false, so we shade the side without the point \((0, 0)\text{.}\)

We can see the correct answer is C.
