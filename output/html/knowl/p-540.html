The solution set is \(\left\{x: \frac{x}{3} \lt 0\right\}\text{.}\) The expression \(\frac{x}{3}\) is defined for all real numbers \(x\text{,}\) and equals zero only if \(x = 0\text{.}\) The remaining real numbers are partitioned into two intervals: \((-\infty, 0)\) and \((0, \infty)\text{.}\) We need to pick a test point in each one and be careful about what happens at \(x = 0\text{.}\)
