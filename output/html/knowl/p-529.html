In this section we discuss graphing the solution to inequalities like \(y > \frac{x+1}{x-1}\) or, more generally, \(y > f(x)\text{,}\) where \(f(x)\) is a rational function. The graph of \(y = f(x)\text{,}\) together with the vertical lines at the points where \(f\) is undefined, partition the plane into several regions. As before, in each region either all points satisfy the inequality or none do. Thus our method is:

  • Find the domain of \(f\text{.}\)

  • Sketch the graph of \(y = f(x)\text{.}\) Use a solid line if the points on the graph belong to the solution set, and a dashed line if not.

  • Pick a test point in each region and determine the regions for which every point belongs to the solution set, then shade these regions on the graph.
