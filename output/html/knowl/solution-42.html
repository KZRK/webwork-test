Solution 3.0.68.1.

To graph the inequality \(x-2y \leq 2\text{,}\) we first graph the line \(x-2y=2\text{.}\) You can use either the table method or slope-triangle method to locate two points on the line.

Note that the line is solid, because the inequality symbol is \(\leq\text{.}\)

Next, we need to decide which side to shade. We choose \((0,0)\) as the test point. We substitute \(x=0\) and \(y=0\) into the inequality into \(x-2y \leq 2\text{,}\) and we have:

\begin{equation*} \begin{aligned}[t] x-2y \amp \leq 2 \\ 1 \cdot 0 - 2 \cdot 0 \amp \leq 2 \\ 0 \amp \leq 2 \end{aligned} \end{equation*}

This is true, so we shade the side with the point \((0, 0)\text{.}\)

We can see the correct answer is C.
