This exercise is concerned with operations with inequality and interval notations.

Match the sets and the inequalities by placing the letter of the inequality next to each set listed below:

  1. \(x\) is nonnegative

  2. All \(x\) in the interval \([-30,4)\)

  3. All \(x\) in the interval \((-30,4]\)

  4. \(x\) is greater than \(-30\)

  1. \(\displaystyle -30\lt x\)

  2. \(\displaystyle -30\leq x\lt 4\)

  3. \(\displaystyle -30\lt x\leq 4\)

  4. \(\displaystyle x\geq0\)
