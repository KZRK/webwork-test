A quadratic function is a function \(f(x) = ax^2+bx+c\text{,}\) where \(a \neq 0\) (otherwise the function is linear, not quadratic). The graph of a quadratic function is a parabola. FigureÂ 3.3.1 shows the graph of \(y = x^2\) on the left and \(y = -x^2\) on the right. The important point is that when \(a > 0\) then the graph of \(y = f(x)\) is shaped like the graph of \(y = x^2\text{,}\) and when \(a \lt 0\) it is shaped like the graph of \(y = -x^2\text{.}\)
