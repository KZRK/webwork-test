Addition, subtraction, multiplication, and division of fractions that involve a variable are all done in the same way they were with real numbers, except that, in addition to the restrictions from above, we also need to restrict the values that the variable can take so that the operations are defined as well. For example, \(\frac{1}{x}\) and \(\frac{x+2}{x-1}\) are defined only for \(x\neq0,1\text{,}\) and \(\frac{x+2}{x-1}=0\) for \(x=-2\text{,}\) so \(\frac{1}{x}\div\frac{x+2}{x-1}\) only makes sense for \(x\neq-2,0,1\text{.}\)
