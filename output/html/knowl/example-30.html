Example 3.1.4.

Graph the solution set to \(\frac{1}{3x-1} \lt \frac{2x - 5}{x + 4}\) on the real number line.

We have \(\frac{1}{3x-1} \lt \frac{2x - 5}{x + 4}\) if and only if

\begin{equation*} \begin{array}{rcl} 0 \amp \lt \amp \frac{2x - 5}{x+ 4} - \frac{1}{3x-1}\\ \\ \amp = \amp \frac{(2x - 5)(3x-1)}{(x+ 4)(3x-1)} - \frac{x+4}{(x+4)(3x-1)}\\ \\ \amp = \amp \frac{(2x - 5)(3x-1) - (x+4)} {(x+4)(3x-1)}\\ \\ \amp = \amp \frac{6x^2 -18x + 1}{(x+4)(3x-1)} \end{array} \end{equation*}

Thus, the solution set is \(\left\{x: \frac{6x^2 -18x + 1}{(x+4)(3x-1)} > 0\right\}\text{.}\)

The expression \(\frac{6x^2 -18x + 1}{(x+4)(3x-1)}\) is undefined when \(x=-4\) and when \(x = \frac{1}{3}\text{.}\) It equals zero when \(6x^2 -18x + 1 = 0\text{,}\) that is, when

\begin{equation*} x = \frac{-(-18) \pm \sqrt{(-18)^2 - 4\cdot6\cdot1} } {2 \cdot 6} = \frac{18 \pm \sqrt{300}}{12} = \frac{18 \pm 10\sqrt{3}}{12} = \frac{9 \pm 5\sqrt{3}}{6} \end{equation*}

Now, \(\frac{9 - 5\sqrt{3}}{6} \approx 0.06\) and \(\frac{9 + 5\sqrt{3}}{6} \approx 2.94\text{.}\) Thus the numbers \(-4, \frac{9 - 5\sqrt{3}}{6}, \frac{1}{3}\) and \(\frac{9 + 5\sqrt{3}}{6}\) (in this order) partition the other real numbers into five intervals. We need to pick a test point in each one of them, and then take our knowledge of what happens at \(-4, \frac{9 - 5\sqrt{3}}{6}, \frac{1}{3}\) and \(\frac{9 + 5\sqrt{3}}{6}\) into account.

For the interval \((-\infty, -4)\) we choose \(x = -5\) as the test point. When \(x = -5\text{,}\) the expression evaluates to \(15.0625\text{,}\) so all points in this interval belong to the solution set. Since the expression is undefined at \(x = -4\text{,}\) it does not belong to the solution set.

For the interval \((-4, \frac{9 - 5\sqrt{3}}{6})\) we choose \(x=-1\) as the test point. When \(x = -1\text{,}\) the expression evaluates to \(-\frac{29}{12} \lt 0\text{,}\) so no point in this interval belongs tp the solution set. Since the expression equals zero when \(\frac{9 - 5\sqrt{3}}{6}\text{,}\) this point is not in the solution set.

For the interval \((\frac{9 - 5\sqrt{3}}{6}, \frac{1}{3})\) we choose \(x = 0.1\) as the test point. When \(x = 0.1\) the expression evaluates to approximately \(0.26 > 0\text{,}\) so all points in this interval belong to the solution set. Since the expression is undefined at \(x = \frac{1}{3}\text{,}\) it does not belong to the solution set.

For the interval \((\frac{1}{3}, \frac{9 + 5\sqrt{3}}{6})\) we choose \(x=1\) as the test point. When, \(x = 1\) the expression evaluates to \(-1.1 \lt 0\text{,}\) so no points in this interval belong to the solution set. Since the expression equals zero when \(\frac{9 -+5\sqrt{3}}{6}\text{,}\) this point is not in the solution set.

For the interval \((\frac{9 + 5\sqrt{3}}{6}, \infty)\text{,}\) we choose \(x = 10\) as the test point. When, \(x = 10\) the expression evaluates to approximately \(1.04 > 0\text{,}\) so all points in this interval belong to the solution set.

The solution set is therefore \((-\infty, -4) \cup (\frac{9 - 5\sqrt{3}}{6}, \frac{1}{3}) \cup (\frac{9 + 5\sqrt{3}}{6}, \infty)\text{.}\) Its graph is shown below.
