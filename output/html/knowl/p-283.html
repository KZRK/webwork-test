In questions 1 - 5, use the rules of exponents to find an expression equivalent to given expression.

  1. \(\displaystyle 36^\frac{1}{2}\)

  2. \(\displaystyle (-125)^\frac{1}{3}\)

  3. \(\displaystyle (\frac{9}{4})^\frac{1}{2}\)

  4. \(\displaystyle x^\frac{1}{4}x^{-\frac{1}{5}}\)

  5. \(\displaystyle (m^\frac{1}{2}n^{-\frac{1}{3}})^{-\frac{1}{6}}\)

  6. Find \(x\) if \(x^\frac{1}{5} = \left(\frac{2}{3}\right)^\frac{1}{2}\text{.}\)
