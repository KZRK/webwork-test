For the interval \((-\infty, -4)\) we choose \(x = -5\) as the test point. When \(x = -5\text{,}\) the expression evaluates to \(15.0625\text{,}\) so all points in this interval belong to the solution set. Since the expression is undefined at \(x = -4\text{,}\) it does not belong to the solution set.
