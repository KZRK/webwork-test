SOLUTION:

Evaluating \(x^2+y^2 = 1\) for \(y = \frac{11}{14}\) gives

\begin{equation*} {(\frac{11}{14}})^2+x^2 = 1 \end{equation*}

Solving for x yields

\begin{equation*} x^2 = {(0.618589574131742)}^2 \end{equation*}

Since the point is in quadrant II, then x is negative and so \(x = -0.618589574131742\text{.}\)
