Solution 3.0.65.1.

We substitute \(y\) values from each ordered pair into the inequality \(y \le -2\) and see whether the inequality is true.

  1. The ordered pair is \((0,0)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} -2 \\ 0 \amp \stackrel{?}{\le} -2 \end{aligned}\)

    Since this inequality is not true, \((0,0)\) is not a solution to the inequality.

  2. The ordered pair is \((2,-5)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} -2 \\ -5 \amp \stackrel{?}{\le} -2 \end{aligned}\)

    Since this inequality is true, \((2,-5)\) is a solution to the inequality.

  3. The ordered pair is \((-3,-2)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} -2 \\ -2 \amp \stackrel{?}{\le} -2 \end{aligned}\)

    Since this inequality is true, \((-3,-2)\) is a solution to the inequality.

  4. The ordered pair is \((6,9)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} -2 \\ 9 \amp \stackrel{?}{\le} -2 \end{aligned}\)

    Since this inequality is not true, \((6,9)\) is not a solution to the inequality.
