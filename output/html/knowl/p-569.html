Look at \((-49)^\frac{4}{2} = \left((-49)^4\right)^\frac{1}{2} = \sqrt{(-49)^4} = \sqrt{5764801} = 2401\text{.}\) We can not compute as \(\left((-49)^\frac{1}{2}\right)^4\) because \((-49)^\frac{1}{2}\) is not defined. We can compute as \((-49)^\frac{4}{2} = (-49)^2 = 2401\) because \((-49)^\frac{4}{2}\) is positive (because \((-49)^4\) is positive) and so is \((-49)^2\text{.}\)
