Example 2.4.3.

Find \(x\) if \(x^\frac{1}{3} = \left(\frac{5}{4}\right)^\frac{1}{7}\text{.}\)

Since \(x^\frac{1}{3} = \left(\frac{5}{4}\right)^\frac{1}{7}\text{,}\) we know that

\begin{equation*} x = \left(x^\frac{1}{3}\right)^{3} = \left(\left(\frac{5}{4}\right)^\frac{1}{7}\right)^{3} = \left(\sqrt[7]{\frac{5}{4}}\right)^{3} = \sqrt[7]{\frac{125}{64}}\text{.} \end{equation*}
