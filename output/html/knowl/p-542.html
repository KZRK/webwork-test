By choosing the test point \(x = -1\) we obtain \(-\frac{1}{3} \lt 0\text{,}\) so every number in \((-\infty, 0)\) is in the solution set. Since \(\frac{0}{3} = 0\text{,}\) the endpoint 0 is not in the solution set. By choosing the test point \(x = 1\) we obtain \(\frac{1}{3} > 0\text{,}\) so no number in \((0, \infty)\) is in the solution set.
