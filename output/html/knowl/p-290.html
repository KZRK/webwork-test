Let \(D\) be a set of real numbers. Any way of associating exactly one real number with each element of \(D\) is called a function. The set \(D\) is called the function's domain. For any \(x\) in \(D\text{,}\) we call \(f(x)\) the value of \(f\) at \(x\). The set \(\{f(x) : x \mbox{ is in } D\} = \{y: y = f(x) \mbox{ for some }\)x\(\mbox{ in } D\}\) is called the function's range â€” that is, the range of \(f\) is the set of all numbers that occur as values of \(f\text{.}\)
