Solution 3.1.3.1.

Notice that \(\frac{6}{x} \lt \frac{3}{4}\) if and only if \(\frac{6}{x} - \frac{3}{4} \lt 0\text{.}\) Now,

\begin{equation*} \begin{array}{rcl} 0 \amp > \amp \frac{6}{x} - \frac{3}{4}\\ \\ \amp = \amp \frac{6}{x} - \frac{3}{4}\\ \\ \amp = \amp \frac{6\cdot4}{4x} - \frac{3x}{4x}\\ \\ \amp = \amp \frac{24-3x}{4x} \end{array} \end{equation*}

Thus the solution set is \(\left\{ x: \frac{24-3x}{4x} \lt 0\right\}\text{.}\) The expression \(\frac{24-3x}{4x}\) is undefined when \(x = 0\text{,}\) and equals zero when \(x = 8\text{.}\) The remaining real numbers are partitioned into three intervals, \((-\infty, 0)\text{,}\) \((0, 8)\) and \((8, \infty)\text{.}\) We must pick a test point in each one, and then be careful about what happens at \(0\) and \(8\text{.}\)

Testing with \(x = -1\) gives \(\frac{24-3 (-1)}{4 (-1)} = -\frac{27}{4} \lt 0\text{,}\) so every number in \((-\infty, 0)\) belongs to the solution set. The expression is undefined at \(x = 0\text{,}\) so it does not belong to the solution set.

Testing with \(x = 1\) gives \(\frac{24-3(1)}{4(1)} = \frac{21}{4} > 0\text{,}\) so no number in \((0, 8)\) belongs to the solution set. Since \(\frac{24-3x}{4x}\) equals zero when \(x = 8\text{,}\) the endpoint \(8\) is not in the solution set.

Testing with \(x = 10\) gives \(\frac{24-3(10)}{4(10)} = -\frac{6}{40} \lt 0\text{,}\) so every number in \((8, \infty)\) belongs to the solution set.
