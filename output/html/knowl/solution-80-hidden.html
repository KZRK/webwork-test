To graph the inequality \(y \geq x\text{,}\) we first graph the line \(y=x\text{.}\) You can use either the table method or slope-triangle method to locate two points on the line.

Note that the line is solid, because the inequality symbol is \(\geq\text{.}\)

Next, we need to decide which side to shade. Usually we substitute \(x=0\) and \(y=0\) into the inequality. However, for this problem, \((0,0)\) is on the line \(y = x\text{,}\) so we have to pick another point.

Let's pick the point \((0, 1)\text{.}\) We substitute \(x=0\) and \(y=1\) into \(y \geq x\text{,}\) and we have:

\begin{equation*} \begin{aligned}[t] y \amp \geq x \\ 1 \amp \geq 1 \cdot 0 \\ 1 \amp \geq 0 \end{aligned} \end{equation*}

This is true, so we shade the side with the point \((0, 1)\text{.}\)

We can see the correct answer is A.
