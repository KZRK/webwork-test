Item.

\(\big(\) and \big) are used to indicate that the endpoints of the interval are not included. These correspond to \(\lt\) and \(>\) when the interval is described using inequalities. For example, \(\big( a, b \big) = \{ x: a \lt x \lt b\}\) . Therefore, \(\big( 2, 5 \big)\) = \(\{ x : 2 \ \textless \ x \ \textless\ 5 \}\text{.}\) This set contains numbers such as 2.0000000001, 4, \(\pi\text{,}\) \(\frac{7}{3}\) etc. There are also infinitely many members of this set.
