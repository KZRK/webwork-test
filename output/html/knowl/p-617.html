In this section we discuss graphing the solution set to an inequality like \(y \leq f(x)\text{,}\) or \(y \geq f(x)\text{,}\) where \(f\) is a function. The solution set is the set of all points for which the inequality is true. The material in this section is very similar to our previous work on graphing the solution set to inequalities involving rational expressions.
