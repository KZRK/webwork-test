Informally, if \(f\) and \(g\) are inverses, then \(g\) “undoes” \(f\) by sending \(y\) back to \(x\text{.}\) Similarly, \(f\) “undoes” \(g\text{.}\) A more formal way to state this involves function composition: if \(f\) and \(g\) are inverses, then

\begin{equation*} (g \circ f)(x) = g(f(x)) = g(y) = x \end{equation*}

by definition, and similarly

\begin{equation*} (f \circ g)(y) = f(g(y)) = f(x) = y\text{.} \end{equation*}
