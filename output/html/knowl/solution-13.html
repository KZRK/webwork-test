Solution 3.0.65.1.

We substitute \(y\) values from each ordered pair into the inequality \(y \le 0\) and see whether the inequality is true.

  1. The ordered pair is \((-3,2)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} 0 \\ 2 \amp \stackrel{?}{\le} 0 \end{aligned}\)

    Since this inequality is not true, \((-3,2)\) is not a solution to the inequality.

  2. The ordered pair is \((0,0)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} 0 \\ 0 \amp \stackrel{?}{\le} 0 \end{aligned}\)

    Since this inequality is true, \((0,0)\) is a solution to the inequality.

  3. The ordered pair is \((3,0)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} 0 \\ 0 \amp \stackrel{?}{\le} 0 \end{aligned}\)

    Since this inequality is true, \((3,0)\) is a solution to the inequality.

  4. The ordered pair is \((-6,-8)\text{.}\)

    \(\begin{aligned}[t] y \amp \stackrel{?}{\le} 0 \\ -8 \amp \stackrel{?}{\le} 0 \end{aligned}\)

    Since this inequality is true, \((-6,-8)\) is a solution to the inequality.
