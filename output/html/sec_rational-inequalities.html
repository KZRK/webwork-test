Section 3.1 Inequalities Involving Rational Expressions

Let's start with an example: Represent the set \(\left\{x:\frac{x+1}{x-5} \geq 0\right\}\) graphically on the real number line.

For each real number \(x\) in its domain, the expression \(\frac{x+1}{x-5}\) is either positive, negative, or zero. We are interested in the values of \(x\) for which it is greater than or equal to zero. The expression can change sign only at places where it equals zero, or at places where it is undefined. Notice that \(\frac{x+1}{x-5}=0\) when \(x=-1\text{,}\) and is undefined when \(x=5\text{.}\) There are no other roots or places where \(\frac{x+1}{x-5}\) is undefined, so \(-1\) and \(5\) are the only places that \(\frac{x+1}{x-5}\) could change sign.

The numbers \(-1\) and \(5\) partition the set of other numbers in \(\mathbb{R}\) into three intervals: \((-\infty, -1)\text{,}\) \((-1, 5)\text{,}\) and \((5, \infty)\text{.}\) Because the expression can change sign only at \(-1\) or \(5\text{,}\) for each of these, either every number in the interval belongs to the set or no number in the interval belongs to the set. We need to determine which possibility occurs for each of them. We can do this by testing one number from each interval and then remembering that we know what happens with the expression at \(-1\) and \(5\text{.}\)

For the numbers less than \(-1\text{,}\) that is the numbers is \((-\infty, -1)\text{,}\) we choose \(-2\) as a test point (any number less than \(-1\) will do, so we can choose one that's easy to calculate with): \(\frac{-2+1}{-2-5}=\frac{-1}{-7}=\frac{1}{7}>0\text{,}\) so \(-2\) is in the set. Thus every number in the interval \((-\infty, -1)\) is in the set. And we need to be careful at the endpoint, \(x = -1\text{.}\) We know already that \(\frac{1-1}{1-5}=0\text{,}\) so the endpoint \(-1\) is in the set.

For the numbers between -1 and 5, that is in \((-1, 5)\text{,}\) we choose 0 as our test point: \(\frac{0+1}{0-5}=\frac{1}{-5}=-\frac{1}{5}\lt 0\text{,}\) so no number in the interval \((-1,5)\) is in the set. Again, we need to be careful at the endpoints (we already know what happens with \(x = -1\)). We know already that \(\frac{5-1}{5-5}\) is undefined, so the endpoint \(5\) is not in the set.

For the numbers greater than 5, that is in \((5, \infty)\text{,}\) we choose 6 as our test point: \(\frac{6+1}{6-5}=\frac{7}{1}=7>0\text{,}\) so every number in the interval \((5,\infty)\) is in the set. We already know what happens at the endpoint 5, and there are no other endpoints to check.

Therefore, all numbers in \((-\infty,-1]\) and \((5,\infty)\) are in the set and no number in \((-1,5]\) is in the set. That is, the set is \((-\infty,-1] \cup (5,\infty)\text{.}\) Its graph on the real number line is shown below.

Example 3.1.1.

{ Represent the set \(\left\{x:\frac{3x+1}{x+4} \geq 1\right\}\) graphically.}

Notice that \(\frac{3x+1}{x+4} \geq 1\) if and only if \(\frac{3x+1}{x+4} -1 \geq 0\text{,}\) which we can rewrite further by finding a common denominator:

\begin{equation*} \begin{array}{rcl} 0 \amp \leq \amp \frac{3x+1}{x+4} -1 \\ \\ \amp =\amp \frac{3x+1}{x+4} - \frac{x+4}{x+4} \\ \\ \amp =\amp \frac{3x+1 - (x+4)}{x+4} \\ \\ \amp =\amp \frac{2x-3}{x+4}. \end{array} \end{equation*}

Therefore, \(\left\{x:\frac{3x+1}{x+4} \geq 1\right\} = \left\{x:\frac{2x-3}{x+4} \geq 0 \right\}\text{.}\) Now we can see that \(\frac{2x-3}{x+4}=0\) when \(x=\frac{3}{2}\) and is undefined when \(x=-4\text{.}\) There are no other roots or places where \(\frac{2x-3}{x+4}\) is undefined, so \(\frac{3}{2}\) and \(-4\) are the only places where \(\frac{2x-3}{x+4}\) could change sign. This partitions the other numbers in \(\mathbb{R}\) into three intervals, and again we need to determine which ones contain numbers that are in the set, and be careful about \(-4\) and \(-\frac{3}{2}\text{.}\)

For \((-\infty, -4)\text{,}\) test \(x = -5\text{:}\) \(\frac{2(-5)-3}{-5+4}=\frac{-13}{-1}=13>0\text{,}\) so \(-5\) is in the set. Therefore every number in the interval \((-\infty,-4)\) is in the set. We know already that \(\frac{2(-4)-3}{-4+4}\) is undefined, so the endpoint \(-4\) is not in the set.

For \((-4, \frac{3}{2})\text{,}\) test \(x = -2\text{:}\) \(\frac{2(-2)-3}{-2+4}=\frac{-7}{2}=-\frac{7}{2}\lt 0\text{,}\) so \(-2\) is not in the set and so no number in the interval \(\left(-4,\frac{3}{2}\right)\) is in the set. We know that \(\frac{2\frac{3}{2}-3}{\frac{3}{2}x+4}=0\text{,}\) so \(x=\frac{3}{2}\) is in the set.

For \((\frac{3}{2}, \infty)\text{,}\) test \(x = 2\text{:}\) \(\frac{2(2)-3}{2+4}=\frac{1}{6}>0\text{,}\) so \(2\) is in the set. Therefore every number in the interval \(\left(\frac{3}{2},\infty\right)\) is in the set.

Therefore the numbers in \((-\infty,-4)\) and \([\frac{3}{2},\infty)\) are in the set and the numbers in \([-4,\frac{3}{2})\) are not in the set. That is, the set is \((-\infty,-4) \cup [\frac{3}{2},\infty)\text{.}\) Its graph on the real number line is shown below.

Example 3.1.2.

Graph the solution set to \(\frac{x}{3} \lt 0\) on the real number line.

The solution set is \(\left\{x: \frac{x}{3} \lt 0\right\}\text{.}\) The expression \(\frac{x}{3}\) is defined for all real numbers \(x\text{,}\) and equals zero only if \(x = 0\text{.}\) The remaining real numbers are partitioned into two intervals: \((-\infty, 0)\) and \((0, \infty)\text{.}\) We need to pick a test point in each one and be careful about what happens at \(x = 0\text{.}\)

By choosing the test point \(x = -1\) we obtain \(-\frac{1}{3} \lt 0\text{,}\) so every number in \((-\infty, 0)\) is in the solution set. Since \(\frac{0}{3} = 0\text{,}\) the endpoint 0 is not in the solution set. By choosing the test point \(x = 1\) we obtain \(\frac{1}{3} > 0\text{,}\) so no number in \((0, \infty)\) is in the solution set.

Therefore, the solution set to the given inequality is \((-\infty, 0)\text{.}\) Its graph on the real number line is shown below.

Example 3.1.3.

{Graph the solution set to \(\frac{6}{x} \lt \frac{3}{4}\text{,}\) and on the real number line}

Notice that \(\frac{6}{x} \lt \frac{3}{4}\) if and only if \(\frac{6}{x} - \frac{3}{4} \lt 0\text{.}\) Now,

\begin{equation*} \begin{array}{rcl} 0 \amp > \amp \frac{6}{x} - \frac{3}{4}\\ \\ \amp = \amp \frac{6}{x} - \frac{3}{4}\\ \\ \amp = \amp \frac{6\cdot4}{4x} - \frac{3x}{4x}\\ \\ \amp = \amp \frac{24-3x}{4x} \end{array} \end{equation*}

Thus the solution set is \(\left\{ x: \frac{24-3x}{4x} \lt 0\right\}\text{.}\) The expression \(\frac{24-3x}{4x}\) is undefined when \(x = 0\text{,}\) and equals zero when \(x = 8\text{.}\) The remaining real numbers are partitioned into three intervals, \((-\infty, 0)\text{,}\) \((0, 8)\) and \((8, \infty)\text{.}\) We must pick a test point in each one, and then be careful about what happens at \(0\) and \(8\text{.}\)

Testing with \(x = -1\) gives \(\frac{24-3 (-1)}{4 (-1)} = -\frac{27}{4} \lt 0\text{,}\) so every number in \((-\infty, 0)\) belongs to the solution set. The expression is undefined at \(x = 0\text{,}\) so it does not belong to the solution set.

Testing with \(x = 1\) gives \(\frac{24-3(1)}{4(1)} = \frac{21}{4} > 0\text{,}\) so no number in \((0, 8)\) belongs to the solution set. Since \(\frac{24-3x}{4x}\) equals zero when \(x = 8\text{,}\) the endpoint \(8\) is not in the solution set.

Testing with \(x = 10\) gives \(\frac{24-3(10)}{4(10)} = -\frac{6}{40} \lt 0\text{,}\) so every number in \((8, \infty)\) belongs to the solution set.

The solution set is therefore \((-\infty, 0) \cup (8, \infty)\text{.}\) Its graph is shown below.

Example 3.1.4.

Graph the solution set to \(\frac{1}{3x-1} \lt \frac{2x - 5}{x + 4}\) on the real number line.

We have \(\frac{1}{3x-1} \lt \frac{2x - 5}{x + 4}\) if and only if

\begin{equation*} \begin{array}{rcl} 0 \amp \lt \amp \frac{2x - 5}{x+ 4} - \frac{1}{3x-1}\\ \\ \amp = \amp \frac{(2x - 5)(3x-1)}{(x+ 4)(3x-1)} - \frac{x+4}{(x+4)(3x-1)}\\ \\ \amp = \amp \frac{(2x - 5)(3x-1) - (x+4)} {(x+4)(3x-1)}\\ \\ \amp = \amp \frac{6x^2 -18x + 1}{(x+4)(3x-1)} \end{array} \end{equation*}

Thus, the solution set is \(\left\{x: \frac{6x^2 -18x + 1}{(x+4)(3x-1)} > 0\right\}\text{.}\)

The expression \(\frac{6x^2 -18x + 1}{(x+4)(3x-1)}\) is undefined when \(x=-4\) and when \(x = \frac{1}{3}\text{.}\) It equals zero when \(6x^2 -18x + 1 = 0\text{,}\) that is, when

\begin{equation*} x = \frac{-(-18) \pm \sqrt{(-18)^2 - 4\cdot6\cdot1} } {2 \cdot 6} = \frac{18 \pm \sqrt{300}}{12} = \frac{18 \pm 10\sqrt{3}}{12} = \frac{9 \pm 5\sqrt{3}}{6} \end{equation*}

Now, \(\frac{9 - 5\sqrt{3}}{6} \approx 0.06\) and \(\frac{9 + 5\sqrt{3}}{6} \approx 2.94\text{.}\) Thus the numbers \(-4, \frac{9 - 5\sqrt{3}}{6}, \frac{1}{3}\) and \(\frac{9 + 5\sqrt{3}}{6}\) (in this order) partition the other real numbers into five intervals. We need to pick a test point in each one of them, and then take our knowledge of what happens at \(-4, \frac{9 - 5\sqrt{3}}{6}, \frac{1}{3}\) and \(\frac{9 + 5\sqrt{3}}{6}\) into account.

For the interval \((-\infty, -4)\) we choose \(x = -5\) as the test point. When \(x = -5\text{,}\) the expression evaluates to \(15.0625\text{,}\) so all points in this interval belong to the solution set. Since the expression is undefined at \(x = -4\text{,}\) it does not belong to the solution set.

For the interval \((-4, \frac{9 - 5\sqrt{3}}{6})\) we choose \(x=-1\) as the test point. When \(x = -1\text{,}\) the expression evaluates to \(-\frac{29}{12} \lt 0\text{,}\) so no point in this interval belongs tp the solution set. Since the expression equals zero when \(\frac{9 - 5\sqrt{3}}{6}\text{,}\) this point is not in the solution set.

For the interval \((\frac{9 - 5\sqrt{3}}{6}, \frac{1}{3})\) we choose \(x = 0.1\) as the test point. When \(x = 0.1\) the expression evaluates to approximately \(0.26 > 0\text{,}\) so all points in this interval belong to the solution set. Since the expression is undefined at \(x = \frac{1}{3}\text{,}\) it does not belong to the solution set.

For the interval \((\frac{1}{3}, \frac{9 + 5\sqrt{3}}{6})\) we choose \(x=1\) as the test point. When, \(x = 1\) the expression evaluates to \(-1.1 \lt 0\text{,}\) so no points in this interval belong to the solution set. Since the expression equals zero when \(\frac{9 -+5\sqrt{3}}{6}\text{,}\) this point is not in the solution set.

For the interval \((\frac{9 + 5\sqrt{3}}{6}, \infty)\text{,}\) we choose \(x = 10\) as the test point. When, \(x = 10\) the expression evaluates to approximately \(1.04 > 0\text{,}\) so all points in this interval belong to the solution set.

The solution set is therefore \((-\infty, -4) \cup (\frac{9 - 5\sqrt{3}}{6}, \frac{1}{3}) \cup (\frac{9 + 5\sqrt{3}}{6}, \infty)\text{.}\) Its graph is shown below.

We close this section by noting that, despite all appearances, it is not true that the intervals in the partition of the real line determined by the roots of the expression and the points where it is undefined are alternately part of the solution set and not part of the solution set (or vice-versa). To see this, work through finding the solution set of \(\frac{(x-1)^2}{(x-2)^2} \geq 0\text{.}\) The solution set is \((-\infty, 1] \cup [1, 2) \cup (2, \infty) = (-\infty, 2) \cup (2, \infty)\text{,}\) that is, all real numbers except 2.

Subsection 3.1.1 Practice Problems

Find the solution sets for the following expressions then graph them on the real number line.

  1. \(\displaystyle \frac{x}{7} > 0\)

  2. \(\displaystyle \frac{9}{x} \leq \frac{7}{5}\)

  3. \(\displaystyle \frac{x+4}{x-2} \lt 0\)

  4. \(\displaystyle \frac{2x+1}{x-5} \lt 3\)

  5. \(\displaystyle \frac{x+2}{1-x} \geq \frac{x-4}{x+3}\)

Subsection 3.1.2 Solutions

1. \((0,\infty)\)

2. \((-\infty , 0)\) and \(\left[\frac{45}{7}, \infty\right)\)

3. \((-4,2)\)

4. \((-\infty , 5)\) and \(\left(16, \infty\right)\)

5. \(\left(-3,1\right)\)
